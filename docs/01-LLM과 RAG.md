# LLM과 RAG

## LLM의 한계점
▪️ 학습 마감일 문제 (지식 컷오프 문제)
    - LLM은 특정 시점까지의 데이터로 학습됨. '지식 컷오프 (knowledge Cutoff)' 라고 함.
▪️ 환각 (Hallucination)
    - 모르는 정보를 마치 아는 듯 대답하는 현상
    - LLM은 확률적으로 가장 그럴듯한 다음단어를 예측하는 방식이기 때문.
▪️ 개인화 불가
    - 일반적인 특성에 대한 대답을 잘 구사하지만, 사내자료, 도메인 적인 지식 등에 대한 대답은 불가함.

## RAG 란
▪️ Retrieval-Augmented Generation '검색 증강 생성'
    - Retrieval(검색) : 정보를 먼저 찾고
    - Augmented(증강) : 그 정보로 LLM을 보강하고
    - Generation(생성) : 답변을 생성함
-> 즉, RAG는 검색하고 이를 기반으로 답변을 생성함

- 기술 문서 등
