# 온톨로지란

## ⚫️ 온톨로지란?

온톨로지는 특정 분야에 존재하는 개념이나 관계를 명확하게 정의하고 구조화한 지식 모델이다.  </br>
인간은 경험적으롬 맥락을 이해하지만 AI는 현실 세계의 구조를 온전히 파악하지 못한다. </br>
온톨로지는 이러한 간극을 줄이기 위해서 AI가 일관된 세계 속에서 개념을 해석하고 추론하도록 함.

### ▪️ 온톨로지 구성요소

| 온톨로지 구성 요소 | 설명 | 예시 |
| ------- | ---- | --- |
| 개체 (Entity) | 도메인 안에서 실제로 존재하고 식별할 수 있는 구체적 대상 | 배달원(이지수), 특정배달주문(주문번호) |
| 클래서 (class) | 유사한 개체를 묶어둔 범주, 도메인의 기본 분류 체계 | 배달원 주문 이라는 class |
| 속성(property) | 클래스에 속한 개체가 공통으로 가진 고유한 특징, 상태 | 배달원의 현재 위치, 근무시간 등, 주문의 정보, 주소, 담당자 등 |
| 관계 (relationship) | 두 개체의 의미있는 연결 | 배달원은 직원이다. 주문은 배달주소를 가지고 있다. | 
| 제약조건 (constraint) | 속성과 관계가 지켜야하는 규칙 -> 도메인의 일관성을 지켜줌 | 한 배달원은 두 개의 주문을 수행할 수 없다 |
